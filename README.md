# 💇‍♀️ Beauty Center Landing Page

Landing page moderna desenvolvida em **React + TailwindCSS + Framer Motion** para apresentar os serviços do **Beauty Center**, um salão de beleza em Rio Branco - AC.

O projeto foi desenvolvido com foco em:

- conversão de clientes
- design elegante
- integração com WhatsApp
- exibição de reels e resultados reais
